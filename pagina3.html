Agrupación

  • Párrafos
    • Define una parte que debe mostrarse como un párrafo en un documento.
    La forma de utilizarlo es:
    <p>
    La forma de cerrar una etiqueta es:
    </p>

  • Texto preformateado
    • Indica que su contenido esta preformateado y que este formato debe ser preservado.
    Para abrir la etiqueta hay que poner:
    <pre>
    Para cerrar:
    </pre>

  • Linea horizontal
    • Representa un quiebre temático entre parrafos de una sección o articulo o cualquier contenido.
    Para poner la linea horizontal hay que escribir:
    <hr>
    Con esto se hace una linea horizontal desde el margen izquierdo hasta el margen derecho

  • Listas
    1. Lista desordenada
      2. Define una lista de artículos sin orden y se declara:
      <ul>
      Para cerrarla hay que poner:
      </ul>
    2. Lista ordenada
      3. Define una lista ordenada de artículos y se declara:
      <ol>
      Para cerrarla hay que poner:
      </ol>
      Las listas ordenadas se pueden representar de diferentes formas:
      1. Ordenada con letras mayúsculas:
        2. <ol TYPE=A>
        <li> Primera línea</li>
        <li> Segunda línea</li>
        <li> Tercera línea</li>
        </ol>
        Y se mostraría así:
        1. Primera línea
        2. Segunda línea
        3. Tercera línea
          4.

      2. Ordenada con letras minúsculas:
        3. <ol TYPE=a>
        <li> Primera línea</li>
        <li> Segunda línea</li>
        <li> Tercera línea</li>
        </ol>
        Y se mostraría así:
        1. Primera línea
        2. Segunda línea
        3. Tercera línea

      3. Ordenada con números:
        4. <ol TYPE=1>
        <li> Primera línea</li>
        <li> Segunda línea</li>
        <li> Tercera línea</li>
        </ol>
        Y se mostraría así:
        1. Primera línea
        2. Segunda línea
        3. Tercera línea

      4. Ordenada con números romanos en mayúscula:
        5. <ol TYPE=I>
        <li> Primera línea</li>
        <li> Segunda línea</li>
        <li> Tercera línea</li>
        </ol>
        Y se mostraría así:
        1. Primera línea
        2. Segunda línea
        3. Tercera línea

      5. Ordenada con números romanos en minúscula:
        6. <ol TYPE=i>
        <li> Primera línea</li>
        <li> Segunda línea</li>
        <li> Tercera línea</li>
        </ol>
        Y se mostraría así:
        1. Primera línea
        2. Segunda línea
        3. Tercera línea

      6. También se puede ordenar del revés:
        7. <ol reversed>
        <li> Primera línea</li>
        <li> Segunda línea</li>
        <li> Tercera línea</li>
        </ol>
        Y se mostraría así:
        1. Primera línea
        2. Segunda línea
        3. Tercera línea

      7. Se puede empezar la lista desde el número que determinemos:
        8. <ol start=5>
        <li> Primera línea</li>
        <li> Segunda línea</li>
        <li> Tercera línea</li>
        </ol>
        Y se mostraría así:
        1. Primera línea
        2. Segunda línea
        3. Tercera línea

    3. Lista de definiciones
      4. Define una lista de definiciones, es decir, una lista de términos y sus definiciones asociadas y se declara:
      <dl>
      Para cerrarla hay que poner:
      </dl>
      Dentro de ella hay dos tipos una etiqueta para el término y una etiqueta para la definición:
      <dt></dt>
      Representa un término definido por el siguiente. <dd>
      <dd></dd>
      Representa la definición de los terminos listados antes que él. Estos terminos han sido listados por el <dt>
