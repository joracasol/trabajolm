Hiperenlace

  • Hiperenlace interno
    • Se realiza dentro de la misma página.
    Primero se realiza un identificador del elemento:
    <a id=elementoX>Elemento X</elemento>
    A continuación se realiza un enlace a ese elemento y haciendo click nos lleva a esa parte de la página:
    <a href=#elementoX>Elemento X</a>
    Para subir a la parte de arriba de la página se le da click y ya nos sube:
    <a href=#top>Ir arriba</a>
  • Hiperenlace absoluto
    • Conduce a una ubicación externa al sitio en el que se encuentra el documento. La ubicación es en Internet, en este caso hay que empezar la referencia por http:// el nombre del dominio y algunas veces el nombre de la página. Si no se escribe el nombre de la página se cargará la página de inicio asociada al dominio.
    Se realiza así:
    <a href="http://www.google.es"">"Visita Google"</a>
  • Hiperenlace relativo
    • Conduce a un documento situado dentro del mismo sitio que el documento actual. Un sitio web es un conjunto de archivos y carpetas, relacionados entre sí, con un diseño similar o un objetivo común, estando todos ellos dentro de una misma carpeta, conocida como carpeta raíz del sitio.
    • Si es en el mismo directorio (directorio raiz) sería así:
      • <a href= "pagina.html">"Enlace página en el directorio raiz"</a>
    • Si es un subdirectorio (dentro del directorio raiz si no no funciona) sería así:
      • <a href= "subdirectorio/pagina.html">"Enlace página en el directorio raiz"</a>
    • Si es un directorio anterior sería así:
      • <a href= "../pagina.html">"Enlace página en el directorio raiz"</a>
  • Hiperenlaces en imagenes
    • Se pueden realizar hiperenlaces en imágenes para ello hay que poner esta estructura:
    <a href= "http://www.google.es"><img src="imagen.jpg" alt="google" /></a>
  • Atributos de los hiperenlaces
    • Target Los valores de target mas habituales son:
    •  _blank abre el documento vinculado en una ventana nueva del navegador. También se puede usar _new.
      Aquí les pongo la estructura:
      • <a href= "http://www.google.es" target="_blank">"Enlace a google en pestaña nueva del navegador"</a>
      <a href= "http://www.google.es" target=-"_new">"Enlace a google en pestaña nueva del navegador"</a>
    • _self es la opción predeterminada o por defecto. Abre el documento vinculado en el mismo marco o ventana que el vínculo.(Es como si no pusieramos nada)
      Aquí les pongo la estructura:
      • <a href= "http://www.google.es" target="_self">"Enlace a la página de google en la misma pestaña"</a>
    Title sirve para ponerles enlace a nuestros links:
    <a href= "http://www.google.es" title= "Enlace google">"Enlace a la página de google"</a>
