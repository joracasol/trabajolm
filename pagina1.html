Formato

  • Etiqueta
    • Para diseñar una página en html lo mas utilizado son las etiquetas. Las etiquetas son fragmentos de texto rodeados por corchetes angulares < >, que tienen funciones y usos específicos y se utilizan para escribir código html. Las etiquetas o tags son la forma de escribir código html.
    La forma de abrir una etiqueta es:
     <elemento>
    La forma de cerrar una etiqueta es:
    </elemento>

  • Atributos
    • Los atributos de las etiquetas html sirven para definir detalles de comportamiento o presentación de la etiqueta a los que se les coloca. Se escriben dentro de la propia etiqueta con su nombre de atributo y el valor del atributo entre comillas.
    Un ejemplo es:
    <elemento atributo="valor_del_atributo">
    Para cerrar se cierra como todas las etiquetas es decir:
    </elemento>

  • Estructura del documento
    • Vamos a escribir en el orden que detallo aquí abajo todos los elementos necesarios para tener un html bien formado.
    1. Tipo de documento
      2. Esto es lo primero que se pone en el bloc de notas. Aquí vamos a definir que el documento está bajo el estandar de HTML5 y se declara:
      <!DOCTYPE html>
    2. Inicio
      3. A continuación se le pone un inicio que representa la raíz de un documento HTML. Todos los demás elementos deben ser descendientes de este elemento. A continuación le ponemos un valor que es el idioma en nuestro caso se pondría "es" y se declara:
      <html lang="es">
      Para cerrarlo se pone al final del documento esto:
      </html>
    3. Cabecera
      4. Representa una colección de metadatos acerca del documento, incluyendo enlaces a, o definiciones de, scripts y hojas de estilo. Se abre poniendo:
      <head>
      Dentro se pone:
      <title>Nombre de la página</title>
      Define el título del documento, el cual se muestra en la barra de título del navegador o en las pestañas de página. Solamente puede contener texto y cualquier otra etiqueta contenida no será interpretada.
      <meta charset="utf-8">
      <meta name="author" content="Nombre del autor">
      Estos son solo dos ejemplos del <meta>. Pero se puede utilizar para una infinidad de funciones. Este define los metadatos que no pueden ser definidos usando otro elemento HTML. Esta información puede ser usada por los buscadores
      <link type="text/css" rel="stylesheet" href="hojaestilo.css" media="screen">
      Usada para enlazar JavaScript y CSS externos con el documento HTML actual.
      La etiqueta <link> incluye cuatro atributos cuando enlaza un archivo CSS:
      • type: Indica el tipo de recurso enlazado. Sus valores están estandarizados y para los archivos CSS su valor siempre es text/css
      • rel: Indica el tipo de relación que existe entre el recurso enlazado (en este caso, el archivo CSS) y la página HTML. Para los archivos CSS, siempre se utiliza el valor stylesheet.
      • href: Indica la URL del archivo CSS que contiene los estilos. La URL indicada puede ser relativa o absoluta y puede apuntar a un recurso interno o externo al sitio web.
      • media: Indica el medio en el que se van a aplicar los estilos del archivo CSS. Más adelante se explican en detalle los medios CSS y su funcionamiento.
      Para cerrarlo se pone al final de todas estas declaraciones esto:
        </head>
      • Cuerpo
        • Representa el contenido principal de un documento HTML. Solo hay uno en cada documento. En las siguientes páginas explicaré los elementos que van dentro. Se representa como:
        <body>
        Para cerrarlo se cierra como todas las etiquetas poniendo:
        </body>
  • Resumen
    • Aquí vemos un resumen de la estructura para formar un html:
      <!DOCTYPE html>
      <html lang="es">
        <head>
          <title>Nombre de la página</title>
          <meta charset="utf-8">
          <meta name="author" content="Nombre del autor">
          <link type="text/css" rel="stylesheet" href="hojaestilo.css" media="screen">

        </head>
         <body>
          Aquí estructuraremos el texto de la página web con las etiquetas que les voy a mostrar en las siguientes páginas ya que el <body> es una parte de las secciones.
        </body>
      </html>
