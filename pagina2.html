Secciones

  • Cuerpo
    • Como he explicado antes representa el contenido principal de un documento HTML. Solo hay un elemento <body> en un documento.
    La forma de utilizarlo es:
    <body>
    La forma de cerrar una etiqueta es:
    </body>
    Los elementos que vamos a ir viendo a continuación van dentro del body.

  • Articulo
    • Define contenido autónomo que podría existir independientemente del resto del contenido. Esta etiqueta será usada para representar un contenido específico de nuestra web. Puede representar un artículo, ya sea un artículo de opinión, un entrada en un blog, un comentario en un foro, una noticia, una descarga, o simplemente un contenido estático de una web como el típico quienes somos o a qué nos dedicamos.
    Para abrir la etiqueta hay que poner:
    <article>
    Para cerrar:
    </article>

  • Sección
    • Define una sección en un documento. La etiqueta section es quizás la más difícil de comprender. En principio se usa para representar un bloque de nuestra página que tiene valor semántico, es decir, que aporta un significado a la página.
    Para abrir hay que poner:
    <section>
    Para cerrar la etiqueta escribimos:
    </section>

  • Navegación
    • Define una sección que solamente contiene enlaces de navegación.
    Para abrir hay que poner:
    <nav>
    Para cerrar la etiqueta escribimos:
    </nav>

  • Inciso
    • Define algunos contenidos vagamente relacionados con el resto del contenido de la página. Si es removido, el contenido restante seguirá teniendo sentido.
    Para abrir hay que poner:
    <aside>
    Para cerrar la etiqueta escribimos:
    </aside>
    Click aquí para ver un ejemplo de articulo y sección
  • Títulos
    • Los elementos de títulos implementan seis niveles de títulos de documentos; <h1> es la de mayor y <h6> es la de menor impotancia. Un elemento de cabecera describe brevemente el tema de la sección que introduce.
    Para abrir hay que poner según el nivel del elemento de cabecera que queramos poner una de estas etiquetas:
    <h1> <h2> <h3> <h4> <h5> <h6>
    Para cerrar la etiqueta escribimos:
    </h1> </h2> </h3> </h4> </h5> </h6>
    Click aquí para ver un ejemplo
  • Encabezamiento
    • Define la cabecera de una página o sección. Usualmente contiene un logotipo, el título del sitio Web y una tabla de navegación de contenidos.
    Para abrir hay que poner esta etiqueta:
    <header>
    Para cerrar la etiqueta escribimos:
    </header>
  • Pie de página
    • Define el pie de una página o sección. Usualmente contiene un mensaje de derechos de autoría, algunos enlaces a información legal o direcciones para dar información de retroalimentación.
    Para abrir hay que poner esta etiqueta:
    <footer>
    Para cerrar la etiqueta escribimos:
    </footer>
  • Pie de página
    • Define el pie de una página o sección. Usualmente contiene un mensaje de derechos de autoría, algunos enlaces a información legal o direcciones para dar información de retroalimentación.
    Para abrir hay que poner esta etiqueta:
    <footer>
    Para cerrar la etiqueta escribimos:
    </footer>
  • Dirección
    • Define una sección que contiene información de contacto.
    Para abrir hay que poner esta etiqueta:
    <adress>
    Para cerrar la etiqueta escribimos:
    </adress>
